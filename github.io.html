Заключение

СПИСОК ИСПОЛЬЗОВАННЫХ ИНТЕРНЕТ-РЕСУРСОВ

Цели:

  • Ознакомиться с всемирной компьютерной сетью Интернет и используемыми в ней технологиями;
  • Освоить приемы эффективного поиска необходимой информации в сетевых ресурсах;
  • Получить практический опыт поиска информации по заданной теме;
  • Освоить методы создания Web- документов;
  • Получить практический опыт создания Web- страниц на основе найденных в сети документов;
  • Получить более глубокую теоретическую и практическую подготовку по специальности на основе самостоятельного изучения и обобщения научной, учебной литературы и нормативных документов, а также формирования умений и навыков по работе в сети;
  • Сделать обзор литературы по теме диссертации, используя ресурсы сети Интернет.

Задачи:

  • Рассмотрение общих принципов построения и функционирования Всемирной компьютерной сети Интернет;
  • Рассмотрение общих принципов организации поиска информации в сети Интернет;
  • Поиск информации по заданной теме в сети Интернет;
  • Создание Web-страницы на основе найденных в сети документов.

  • Использование поисковых систем Поисковые системы (например, Google, Yandex, Bing) являются основным инструментом для поиска информации в
    Интернете. Для эффективного поиска важно правильно формулировать запросы и использовать различные операторы поиска, такие как кавычки для
    точной фразы или минус для исключения определенных слов.
  • Использование ключевых .Чем точнее и конкретнее запрос, тем быстрее можно найти нужную информацию. Это предполагает: Четкость в
    формулировке запроса. Уточнение темы с помощью уточняющих слов
  • Применение фильтров поиска Современные поисковые системы и платформы (например, Google или YouTube) предлагают фильтры для уточнения
    поиска: по дате, типу контента (картинки, видео, новости, форумы), языку и региону. Это помогает ограничить поиск и найти информацию
  • Использование специализированных ресурсов и баз данных. Кроме общих поисковых систем, существуют специализированные базы данных и сайты
    для поиска информации по конкретным темам, например: Google Scholar — для поиска научных публикаций. PubMed - для поиска медицинской
    информации.
  • Оценка качества и достоверности информации Важным аспектом поиска является критическая оценка найденных источников. Не вся информация в Интернете достоверна, поэтому необходимо учитывать: Авторитетность источника. Актуальность и свежесть данных.
  • Система категоризации и тегирования Сайты и ресурсы в Интернете часто используют системы тегов, категорий и метаданных для упрощения поиска. Например, на платформах, таких как Wikipedia, Youtube или блоги, информация организована с помощью категорий и ключевых слов, что облегчает навигацию и поиск.

Принципы организации поиска информации в сети Интернет


  • Использование агрегаторов информации .Агрегаторы собирают информацию с разных источников и предоставляют ее в структурированном виде.
  • Постоянное обновление запросов . При поиске информации важно учитывать возможность изменения данных со временем, а также использовать новейшую информацию.

Заключение

В глобальной сети Интернет можно найти информацию по любой интересующей теме. Но самое трудное в работе с сетью Интернет — найти нужную информацию. Поисковые системы значительно облегчают работу в Интернет и помогают нам быстро найти нужную информацию в огромном массиве серверов Интернет.


Наилучшего эффекта поиска информации можно достичь, комбинируя поиск с помощью поисковых систем и поиск по каталогам. Так же для улучшения качества найденной информации можно использовать поиск по синониму искомого слова, а так же поиск похожих документов.


При выполнении данной работы были выполнены поставленные задачи:

  • проанализирована структура и функционирование всемирной компьютерной сети Интернет;
  • раскрыты общие принципы и наиболеее известные способы организации поиска информации в сети Интернет;
  • выполнен аналитический обзор найденных документов;
  • создана Web-страница по теме работы на основе найденных документов.


Таким образом, при поиске информации для магистерской диссертации на тему «Биомеханический анализ движений, как средство профилактики травматизма при занятиях атлетической гимнастикой» были использованы оба способа поиска информации в сети Интернет – поиск в каталогах информационных ресурсов (http://cyberleninka.ru ) и поиск с помощью поисковых машин (http://scholar.google.com , https://freefullpdf.com ).
